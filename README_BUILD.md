# Как собрать CollabStudy.exe

## Требования
- **Node.js** версии 16+ → https://nodejs.org (скачай LTS)
- **Windows 10/11** (для сборки .exe)

## Сборка одной командой

1. Распакуй этот архив в любую папку
2. Запусти `BUILD.bat` двойным кликом
3. Жди 1-2 минуты
4. В папке `dist\` появятся два файла:
   - `CollabStudy Setup 1.0.0.exe` — **установщик** (рекомендуется)
   - `CollabStudy 1.0.0.exe` — **portable** (без установки, просто запустить)

## Или вручную через терминал

```bash
npm install
npm run build:win
```

## Запуск без сборки (для разработки)

```bash
npm install
npm start
```

---
Если возникли проблемы — убедись что Node.js установлен и перезапусти BUILD.bat от имени администратора.
